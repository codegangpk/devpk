# PurpleMap Support Page

## PurpleMap - Save Places That Matter

Welcome to the support center for **PurpleMap**.

PurpleMap helps you save meaningful places on a map, organize them by category, and add photos to remember each location.

Use it for favorite spots.
Use it for memories.
Use it for places you want to find again.
